# task-tracker

## Installation 
`npm install`

## Launching
`npx ts-node src/index.ts`

## Commands
add              — add a new task

update           — update a task by id

delete           — delete a task by id

list             — list all tasks or tasks by category

mark-in-progress — mark the task as in-progress by id

mark-done        — mark the task as done by id

## Usage:
add              `<name>`        | -a   `<name>`,

update           `<id>` `<name>` | -u   `<id>` `<name>`,

delete           `<id>`          | -d   `<id>`,

list             `<status>`      | -l   `<status>`,

mark-in-progress `id>`           | -mip `<id>`,

mark-done        `<id>`          | -md  `<id>`
